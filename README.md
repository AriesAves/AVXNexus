This tool is designed to be a standalone application. It utilizes the browser's native PDF engine within an iframe to provide unrestricted, full-page, zoomable, and scrollable PDF previews without external libraries. It features a simulated AI engine that performs deep analysis on metadata and generates detailed reports, including specific font library matching as requested.

Instructions
Save the code below as index.html.
Open it in any modern web browser (Chrome, Edge, Firefox, Safari).
Drag and drop a PDF or Image file to begin the AI analysis.
